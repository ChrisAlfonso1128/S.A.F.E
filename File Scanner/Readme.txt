# File Scanning Engine

## Overview
Develop the module responsible for scanning directories and collecting file information.

## Responsibilities
- Scan selected folders recursively.
- Read file metadata:
  - Name
  - Path
  - Size
  - Last Modified Date
  - Last Accessed Date
- Skip inaccessible or protected files.
- Return file information in a structured format.

## Deliverables
- File scanner module
- Metadata collection functions
- Error handling for inaccessible files
